# LoPAS — Life-oriented Performance Analysis System

> **"測定可能にすることが、変えられることの前提だ。"**

LoPASは、AI・人間・社会の思考・構造・崩壊を**測定し、設計する**ための
オープンプロトコル＆指標フレームワークです。

---

## 🌏 What is LoPAS?

| 従来のAI | LoPAS |
|----------|-------|
| 即答 | 沈黙を設計する |
| 正解探し | 可能性探し |
| 効率化 | 創造化 |
| 人間の補助 | 人間の拡張 |

LoPASは「思考OS」です。Claude / GPT / Gemini など、モデルに依存しない**哲学レイヤー**。

## 📐 Core Philosophy

**攻め（OFFENSE）:** TRS / DoQ / CCI が高い → 創造・集中・重要判断を前倒し  
**守り（DEFENSE）:** SCI / GFI / AFI が高い → 休息・負荷軽減・構造補修

**CHD（Controlled Hallucination Design）:**  
ハルシネーション＝バグ → 創造エンジン として設計する理論

**PRIME DIRECTIVE:**  
> Never Collapse. Never Isolate.
